# MultiWA Brand Guidelines

## Logo
The MultiWA logo features a green chat bubble with a stylized "W" made from connected nodes, symbolizing WhatsApp messaging and workflow automation.

## Color Palette

### Primary Colors
- **MultiWA Green**: `#25D366` (WhatsApp Green)
  - RGB: 37, 211, 102
  - HSL: 147°, 73%, 49%
  - Usage: Primary brand color, CTAs, active states

- **Dark Green**: `#128C7E`
  - RGB: 18, 140, 126
  - Usage: Hover states, secondary accents

### Neutral Colors
- **White**: `#FFFFFF`
  - Usage: Backgrounds, text on dark
  
- **Light Gray**: `#F5F5F5`
  - Usage: Secondary backgrounds
  
- **Medium Gray**: `#9E9E9E`
  - Usage: Muted text, borders
  
- **Dark Gray**: `#333333`
  - Usage: Primary text

- **Black**: `#1A1A1A`
  - Usage: Headings, dark mode

### Semantic Colors
- **Success**: `#4CAF50`
- **Warning**: `#FFC107`
- **Error**: `#F44336`
- **Info**: `#2196F3`

## Typography
- **Headings**: Inter (Bold, Semibold)
- **Body**: Inter (Regular, Medium)
- **Code**: JetBrains Mono

## Usage Guidelines
1. Always use logo on white or light backgrounds
2. Maintain minimum clear space around logo
3. Don't stretch or distort the logo
4. Use MultiWA Green (#25D366) as primary accent
